## Description

A Nest.js project that replicates the core functionality of the [Honk](https://honk.me/) app

You can find the mobile app source code written in flutter [Here](https://github.com/SanchezQb/honk-clone-flutter)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
